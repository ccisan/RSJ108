How to Create a 4-frame Waving Animation

Completion time: ± 1 hour
Waves hello: infinite

Animation is a skill that is both fun and useful! There are a large variety of styles, programs, and techniques that come in handy when animating. For now, we’re going to be using Procreate’s animation function so we can create a simple 4-frame waving animation together.

  • Things You'll Need

  • Procreate + a very basic understanding of the program
  • iPad
  • Apple Pencil (optional but reccomended)
  • The willingness to learn!

  1. Decide and Draw.

    First things first, you need to decide what you want to animate. I struggle a lot with deciding what I want to create, so I recommend picking something simple at first. A wave can be a complex motion, but it can also be done quickly using transformation tools. Let’s draw our first blank form! Just get the general idea of the body, like a doll.

  2. Duplicate and Transform.

    Now that we have our form, we’re going to duplicate the layer. Using a clipping mask to fill a random different color, then merge the copy with the mask. You’ll end up with the same drawing on top of itself, but in a different color. Then, tap on the “S” shape on the top left. Select the waving arm, then tap the “pointer”. Using the dot at the top, rotate the arm slightly so that the shoulder is in the same place but the hand is slightly off. Repeat for the head and other arm.

  3. Repeat Step 2.

    Repeat the process with your second layer. You should have 3 layers with 3 slightly rotated parts. You can lower the opacity so you can see what you’ve changed: this is called “onion skinning”. My example just shows all layers visible opaquely.

  4. The Fourth Layer.

    Duplicate your second layer to be on top of your third. You should now have 4 layers, with your second layer being used twice as a transition.

  5. Lower Opacity and Redraw with Details.

    Group all the layers together by sliding right on each and selecting “group” by the plus. Individually lower the opacity on each by tapping on the “N” next to each layer. We’re going to create a new layer outside of this group where we will be drawing the more detailed form. Add whatever clothes, hair, etc you want here. I slightly adjusted my head tilt here, and made her a cute girl. Note how I made the hair visible from the front.

  6. Continue Cleanup and Detailing for Final Frames.

    Repeat this process for each frame, but use both the reference and previous frame to see. Slightly extend the hair as the head tilts. You want to have the hair tips visible so you can see where they’re swaying from behind the head. Make sure to re-duplicate layer 2 so you have all 4 frames. Once complete, you can erase the overlapped hair since all frames are complete. Use clipping masks to make all your final frames’ lines black. Merge the clippings to the layers, and make sure you don’t merge your frames together (it happens).

  7. Color.

    Start coloring! I used a warm-primary color palette on a medium-warm skin tone. I used the “artist’s crayon” brush to color because I like the textured, soft look. You can use any brushes you like! Color a layer underneath each frame, then merge the lines with the colors when complete.

  8. Preparing Your Layers for Animation.

    Make sure only your final, colored and lined frames are turned on. Nothing else should be visible. Opening the “wrench” tap at the top left, select “canvas” and turn Animation Assist on. A small timeline should appear at the bottom. If your layers are correct, only 4 frames should be visible. Use the settings in the image below.

  9. Export and Enjoy.

    Export by going to “Actions” (wrench), “Share”, then “Animated GIF”. It should save as a gif to your iPad. You can use either option. Web Ready loads faster on the web, but is lower quality. High Quality looks great but takes more time to load. Here is my final gif, exported as a High Quality!
